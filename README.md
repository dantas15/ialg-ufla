# Intro-alg-ufla

This is a repo to keep my C++ exercises I make during my Alg classes

## Useful commands

- Install `g++`

```bash
sudo apt install g++
```

### Compile with g++

```bash
g++ -Wall -c "file_name.cpp"
```

### Build with g++

```bash
g++ -Wall -o "executable_file_name" "file_name.cpp"
```

### Execute file on Linux

```bash
"./executable_file_name"
```
